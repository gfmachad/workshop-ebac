##  Getting started

Clone the project and access the folder

```bash
$ git clone https://github.com/gfmachad/workshop-ebac.git
```

Follow the steps below

```bash
# Install the dependencies
$ yarn
```

or

```bash
# Start the project
$ yarn start
```
